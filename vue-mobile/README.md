# web-demo

## 介绍

> 该项目是基于H5为前端+express作为服务器的新手入门、进阶级别的项目。

** 技术栈 **
- vue.js
- express
- mongodb

** UI库 **
- element-ui
- vant-ui
- mint-ui
- ...
